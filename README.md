# \<tic-tac-toe>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i tic-tac-toe
```

## Usage
```html
<script type="module">
  import 'tic-tac-toe/tic-tac-toe.js';
</script>

<tic-tac-toe></tic-tac-toe>
```
